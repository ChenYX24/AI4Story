# requirements 环境初始化

- 时间：2026-04-18 11:20 +08
- 目标：复用现有 Miniconda，为项目创建 `ai4story` 环境并安装 `requirements.txt` 依赖。

## 已知事实

- 系统：macOS arm64
- Shell：`/bin/zsh`
- 现状：`/Users/cyx/miniconda3` 已存在，但 `conda` 未在 PATH 中。
- 依赖源：[requirements.txt](../../requirements.txt)
- 环境声明：[environment.yml](../../environment.yml)

## 执行步骤

1. 复用已有 `miniconda3`，不重复安装。
2. 创建 `ai4story` Conda 环境并安装 Python 3.11。
3. 使用 `pip install -r requirements.txt` 安装项目依赖。
4. 将 `conda init zsh` 写入用户 `~/.zshrc`。
5. 验证 `python --version` 与关键包导入。

## 验证命令

- `source "$HOME/miniconda3/etc/profile.d/conda.sh"`
- `conda activate ai4story`
- `python --version`
- `python -c "import rembg, onnxruntime, PIL, requests, tqdm, vtracer; print('imports ok')"`

## 风险

- `onnxruntime`、`rembg` 依赖二进制轮子；若镜像或网络异常，安装可能需要重试。
- `conda init zsh` 会修改用户 `~/.zshrc`。
